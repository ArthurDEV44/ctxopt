# Installation

Distill MCP Server can be installed globally or run directly with npx.

## Prerequisites

- **Node.js 18+** required
- An MCP-compatible IDE (Claude Code, Cursor, or Windsurf)

## Install Methods

### Global Installation (Recommended)

```bash
npm install -g distill-mcp
```

After installation, verify it works:

```bash
distill-mcp --version
distill-mcp doctor
```

### Run with npx

No installation needed - run directly:

```bash
npx distill-mcp serve
```

### From Source

```bash
git clone https://github.com/ArthurDEV44/ctxopt
cd distill
bun install
bun run build
```

## IDE Configuration

After installing, configure your IDE to use the MCP server.

### Claude Code

Edit `~/.claude/mcp.json`:

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp"]
    }
  }
}
```

Or with global install:

```json
{
  "mcpServers": {
    "distill": {
      "command": "distill-mcp",
      "args": ["serve"]
    }
  }
}
```

### Cursor

Edit Cursor's MCP settings (Settings > MCP Servers):

```json
{
  "distill": {
    "command": "npx",
    "args": ["distill-mcp"]
  }
}
```

### Windsurf

Edit `~/.windsurf/mcp.json`:

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp"]
    }
  }
}
```

## Verify Installation

Run the doctor command to verify everything is configured:

```bash
distill-mcp doctor
```

This checks:
- Node.js version
- Package installation
- IDE configurations

## CLI Commands

```bash
distill-mcp serve      # Start the MCP server
distill-mcp setup      # Auto-configure IDEs
distill-mcp doctor     # Check installation
distill-mcp --help     # Show all commands
```
