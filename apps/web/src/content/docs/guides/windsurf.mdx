# Intégration Windsurf

Ce guide vous accompagne dans la configuration de Distill avec l'IDE Windsurf.

## Prérequis

- Windsurf installé ([télécharger](https://codeium.com/windsurf))
- Node.js 18+ installé

## Configuration rapide (30 secondes)

```bash
npm install -g distill-mcp
distill-mcp setup --windsurf
```

Terminé ! Redémarrez Windsurf et vous êtes prêt.

## Configuration manuelle

Si la configuration automatique n'a pas fonctionné, éditez `~/.codeium/windsurf/mcp_config.json` :

```json
{
  "mcpServers": {
    "distill": {
      "command": "distill-mcp",
      "args": ["serve"]
    }
  }
}
```

Ou avec npx (sans installation globale) :

```json
{
  "mcpServers": {
    "distill": {
      "command": "npx",
      "args": ["distill-mcp", "serve"]
    }
  }
}
```

## Vérifier l'installation

Redémarrez Windsurf, puis demandez :

```
Liste les outils MCP disponibles
```

## Dépannage

### MCP ne charge pas

1. Vérifiez l'emplacement du fichier de config : `~/.windsurf/mcp.json`
2. Validez la syntaxe JSON
3. Exécutez `distill-mcp doctor` pour vérifier l'installation

### Erreurs de permissions

Sur macOS, vous devrez peut-être autoriser le terminal dans Préférences Système > Sécurité et confidentialité.
